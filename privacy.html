Политика конфиденциальности

Последнее обновление: 13 апреля 2026

1. Какие данные мы собираем

Мы собираем минимум данных, необходимый для работы сервиса:

  • Telegram ID — для идентификации подписки и доставки ключа
  • Email — только если вы добровольно указали его для резервной копии ключа
  • Данные оплаты — обрабатываются платёжными провайдерами (ЮKassa, CryptoBot), мы не храним данные карт

2. Какие данные мы НЕ собираем

  • Историю посещённых сайтов
  • Содержимое трафика
  • IP-адреса подключений
  • Временные метки сессий
  • Объём переданных данных по пользователям

Логирование трафика отключено на уровне сервера. Мы технически не имеем возможности видеть, какие ресурсы вы посещаете.

3. Как мы используем данные

  • Telegram ID — для управления подпиской, отправки уведомлений об истечении, доставки ключа
  • Email — для отправки резервной копии ключа подключения (если указан)

4. Хранение данных

Данные хранятся на серверах в Нидерландах, за пределами юрисдикции Российской Федерации и альянса Five Eyes. База данных зашифрована, доступ ограничен.

5. Передача данных третьим лицам

Мы не передаём, не продаём и не предоставляем ваши данные третьим лицам, за исключением:

  • Платёжные провайдеры (ЮKassa, CryptoBot) — для обработки оплаты

6. Удаление данных

Вы можете запросить полное удаление ваших данных, обратившись в поддержку: @netcloak_support_bot. Данные будут удалены в течение 72 часов.

7. Файлы cookie

Сайт netcloak.win не использует cookie. Для хранения состояния оплаты используется localStorage браузера, который не передаётся на сервер.

8. Контакты

По вопросам конфиденциальности: @netcloak_support_bot
