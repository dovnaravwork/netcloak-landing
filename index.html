Частые вопросы

Как это работает?
NetCloak шифрует весь ваш интернет-трафик и направляет его через защищённые серверы в Европе. Для вашего провайдера трафик выглядит как обычный HTTPS — неотличим от посещения обычных сайтов.
Вы ведёте логи?
Нет. Мы не храним историю посещений, IP-адреса и данные о трафике. Технически не можем — логирование отключено на уровне сервера.
На каких устройствах работает?
Android (v2rayNG), iPhone/iPad (Streisand), Windows и Mac (Hiddify). Подключение занимает 2 минуты — скопируйте ключ и вставьте в приложение.
Работает ли в России?
Да. Мы используем современные протоколы шифрования, которые не обнаруживаются системами анализа трафика. 2 сервера с автопереключением — если один недоступен, трафик идёт через второй.
Какие способы оплаты?
Банковские карты и СБП (через ЮKassa), криптовалюта USDT/TON (через CryptoBot), Telegram Stars (в боте). Все способы работают из России.
Можно ли вернуть деньги?
Да, в течение 7 дней после оплаты. Напишите в поддержку @netcloak_support_bot — вернём без вопросов.
Какой протокол используется?
VLESS + XHTTP поверх TLS — один из самых устойчивых к анализу трафика протоколов на 2026 год. Для вашего провайдера соединение выглядит как обычное посещение сайта по HTTPS. Два сервера с автопереключением для стабильности.
Чем NetCloak отличается от других?
Не нужно настраивать свой сервер (как Outline/Amnezia self-hosted). Цена от 50₽/мес — в 3–7 раз дешевле коммерческих при том же уровне защиты. Работает когда другие блокируют — протоколы обновляются под текущие методы фильтрации. Оплата из РФ без проблем.

Политика конфиденциальности

Последнее обновление: 13 апреля 2026

1. Какие данные мы собираем

Мы собираем минимум данных, необходимый для работы сервиса:

  • Telegram ID — для идентификации подписки и доставки ключа
  • Email — только если вы добровольно указали его для резервной копии ключа
  • Данные оплаты — обрабатываются платёжными провайдерами (ЮKassa, CryptoBot), мы не храним данные карт

2. Какие данные мы НЕ собираем

  • Историю посещённых сайтов
  • Содержимое трафика
  • IP-адреса подключений
  • Временные метки сессий
  • Объём переданных данных по пользователям

Логирование трафика отключено на уровне сервера. Мы технически не имеем возможности видеть, какие ресурсы вы посещаете.

3. Как мы используем данные

  • Telegram ID — для управления подпиской, отправки уведомлений, доставки ключа
  • Email — для отправки резервной копии ключа (если указан)

4. Хранение данных

Данные хранятся на серверах в Нидерландах, за пределами юрисдикции РФ и альянса Five Eyes. База данных зашифрована, доступ ограничен.

5. Передача данных третьим лицам

Мы не передаём, не продаём и не предоставляем ваши данные третьим лицам, за исключением платёжных провайдеров (ЮKassa, CryptoBot) для обработки оплаты.

6. Удаление данных

Запросите полное удаление через @netcloak_support_bot. Данные удаляются в течение 72 часов.

7. Файлы cookie

Сайт не использует cookie. Для хранения состояния оплаты используется localStorage, который не передаётся на сервер.

8. Контакты

По вопросам конфиденциальности: @netcloak_support_bot

Условия использования

Последнее обновление: 13 апреля 2026

1. Описание сервиса

NetCloak — сервис шифрования интернет-трафика и защиты соединения. Сервис предоставляет ключ подключения для настройки защищённого соединения на устройствах пользователя.

2. Подписка и оплата

  • Доступны тарифы с оплатой на 1, 3, 6 и 12 месяцев
  • Оплата через ЮKassa (карты, СБП), CryptoBot (крипта) и Telegram Stars
  • Подписка не продлевается автоматически
  • Для продления — новая подписка через бот или сайт

3. Пробный период

7 дней, 5 устройств, один раз. Без привязки карты или регистрации.

4. Возврат средств

Возврат в течение 7 дней с момента оплаты через @netcloak_support_bot. Возврат тем же способом оплаты.

5. Допустимое использование

  • Не использовать для незаконной деятельности
  • Не передавать ключ за пределами оплаченного количества устройств
  • Не пытаться нарушить работу серверов

6. Ограничение ответственности

  • Сервис предоставляется «как есть»
  • Цель — uptime 99.9%, без гарантии
  • При недоступности более 72 часов — продление подписки

7. Прекращение доступа

При нарушении условий доступ прекращается с возвратом неиспользованной части.

8. Изменение условий

Актуальная версия — на этой странице. При существенных изменениях уведомим через бот.

9. Контакты

Поддержка: @netcloak_support_bot
